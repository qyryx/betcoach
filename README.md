# BetCoach

BetCoach is a web application that simulates betting company.

## Installation

Install all packages using the package manager [npm](https://www.npmjs.com/). Do this in **both** folders.

```bash
npm install
```

## Usage

```python
# starts both servers on localhost
npm start

```
